CSS

  1. Início (1996)
    O CSS foi proposto por Håkon Wium Lie e introduzido como uma forma de estilizar documentos HTML, permitindo que os desenvolvedores controlassem o layout, as cores e as fontes.
  2. CSS1 (1996)
    A primeira especificação do CSS foi desenvolvida, oferecendo controles básicos sobre a apresentação, como fontes, cores e margens.
  3. CSS2 (1998)
    Introduziu novas funcionalidades como posicionamento, suporte a media queries para diferentes dispositivos e identificação de elementos com seletores.
  4. CSS2.1 (2011)
    Uma atualização que consolidou e corrigiu a versão CSS2, removendo recursos não implementados e estabelecendo uma base mais sólida.
  5. CSS3 (2011 em diante)
    Começou a ser desenvolvido em módulos, trazendo melhorias significativas, como bordas arredondadas, gradientes, sombras, animações e flexbox. Tornou-se uma parte fundamental do design responsivo e moderno.
  6. Desenvolvimento Contínuo
    A especificação CSS continua a evoluir, introduzindo novos recursos e melhorias, como CSS Grid, variáveis CSS e suporte aprimorado para layouts.
